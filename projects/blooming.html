BLOOMING ENGINE

Interactive WebGL AI Generative

Live Interaction: Drag to Rotate / Scroll to Zoom

Technical Narrative

"Blooming Engine" is an experimental WebGL project that merges generative flora with AI-driven interactions.

The core challenge was balancing the visual fidelity of 10,000+ simultaneous particles with the low-latency response of the AI interpreter. Each plant is procedurally generated using noise functions.

Tech Stack

  • Three.js / WebGL
  • GLSL Shaders
  • GPU Instancing
