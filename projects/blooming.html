The Concept

An organic growth simulation where mathematics meet artistic intuition.

The Blooming Engine utilizes Custom GLSL Shaders to calculate the growth of each petal in real-time. By offloading complex vertex animations to the GPU, the system can render thousands of blooming instances while maintaining a smooth 60FPS interaction.

Core Tech

  • Three.js / WebGL
  • GLSL Shader Lab
  • OpenAI API Integration
