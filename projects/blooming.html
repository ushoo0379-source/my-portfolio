Technical Narrative

"Blooming Engine" is an experimental WebGL project that merges generative flora with AI-driven interactions.

By leveraging Three.js and custom GLSL shaders, the project simulates an organic growth system where every bloom is unique. The integration of OpenAI API allows the engine to interpret emotional prompts and translate them into visual parameters such as density, color gradients, and petal velocity.

Tech Stack

  • Three.js / WebGL
  • GLSL Custom Shaders
  • OpenAI GPT-4 API
  • Real-time Physics

Generative Logic

The core challenge was balancing the visual fidelity of 10,000+ simultaneous particles with the low-latency response of the AI interpreter. Each plant is procedurally generated using noise functions, ensuring that no two "blooms" are identical during the live simulation.
