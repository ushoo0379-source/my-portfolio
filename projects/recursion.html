3D Motion Design 2026

Project Showcase: 4K Render Output

Project Narrative

"PIGOUT" 是一个关于视觉张力与物质消耗的探索项目。 通过动态图形展示了从有序到混沌的转变。

在这个项目中，我专注于高精度建模和复杂的材质表现。 所有的动态效果均通过 Houdini 模拟，并在 Redshift 中完成最终渲染，力求还原最真实的物理质感。

Core Competencies

  • Procedural Modeling
  • Dynamic Simulation
  • Look Development
  • Redshift Rendering
