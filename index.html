PIG OUT

Character Animation 2D Rigging

Character Study: Motion & Rigging showcase

Project Narrative

"PIG OUT" focuses on the fluid dynamics of character movement and expressive 2D rigging.

此项目是我在 SVA 期间对角色动画的一次深入尝试。我通过自建的骨骼系统（Rigging）实现了高度灵活的肢体动作，并结合了传统动画的节奏感，展现了一个既夸张又具有生命力的视觉形象。

Core Competencies

  • Character Rigging
  • Keyframe Animation
  • Vector Illustration
  • Visual Storytelling
